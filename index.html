Cozhihu

A chrome extension for zhihu skins

Add Cozhihu to chrome

Currently v0.4.10

If you have a better idea, found some bugs or want to improve this project, please send me an inbox message or create an github issue
Cozhihu gives you a customizable and colorful skins, and which some of them are night mode.

Language support

Cozhihu comes with support for 3 languages built-in.

No tracking data

we don't tracking any user using website private data, protect your privacy.

Multiple themes

Fully skinnable, over 25 themes for you to choose.

Fully extensible

Use sass to write the css, customize color make more faster.

Dynamic preview

Allow users hover the color panel to preview.

Browser support

Support for all chrome-like browser.
